## 🇺🇸 ENG

### Clone😎

```
git clone 
```

### Access the folder🤓

```
cd <folder>
```
### Install dependencies🤠
```
npm install
```
### Start application🤩
```
npm run dev
```
### default port:🤗

```
http://localhost:5000/
```
